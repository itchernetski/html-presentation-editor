Untitled Presentation

Click any text to edit it. Press E to toggle edit mode. Cmd+S to save.

This is a self-contained HTML presentation editor. Everything you see — including this paragraph, the toolbar, and even the editor itself — lives inside one file. Edit it in your browser, save it back to disk, and share the file with anyone.

Three things to try

  • Hover between any two blocks to add a new one
  • Hover a block to move, duplicate, or delete it
  • Press S to flip into slides mode and use arrow keys
